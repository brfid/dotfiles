# projects agent guide

## Purpose

This folder holds public architecture and setup docs for larger systems.
It is context, not a normal live-config sync target.

## Read first

- For repo-wide context, read `../AGENTS.md`.
- target project README files in this folder

## Rules

- Keep docs public-safe.
- Put private identities, secrets, and live data outside this repo.
- Use this folder for system overviews, not runtime state.
- If a project later needs real syncable config, give it its own clear local contract.
