# dotfiles+ agent guide

Read `README.md` first.

This repo stores machine and tool setup notes and reusable LLM agent config (in `agents/`). Local LLM client config is generated by an LLM from this repo; it is not source of truth.

Each root folder owns its setup notes in its own `README.md`.

Keep tracked files safe to publish. Do not add credentials, generated state, machine inventories, job-specific config, private paths, private agent policy, sessions, logs, caches, or databases.
