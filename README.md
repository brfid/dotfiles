# dotfiles

Personal configuration files.

## Install

```bash
git clone https://github.com/brfid/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.sh
```

## Structure

- `nvim/` - Neovim (LazyVim)
- `shell/` - Zsh, environment variables
- `bash/` - Bash
- `tmux/` - Tmux
- `git/` - Git config template
- `scripts/` - Sync automation
