# nvim

LazyVim base plus local overrides for outline, snacks explorer, language servers, icons, and editing behavior.

Live path:
- Copy or symlink this folder to `~/.config/nvim/`.

Notes:
- Runtime files such as `lazy-lock.json` and `lazyvim.json` stay outside git.
- See upstream setup docs at <https://lazyvim.github.io/installation>.
