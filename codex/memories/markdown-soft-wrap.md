# Markdown prose wrapping preference

Prefer soft-wrapped prose in Markdown: keep each prose paragraph as one physical line and let the editor/client wrap visually.

Do not hard-wrap normal prose at a fixed column. Preserve required line breaks for Markdown syntax, including front matter, headings, lists, tables, code fences, block quotes when useful, and other exact blocks.

Name for this preference: soft wrap, unwrapped Markdown paragraphs, or no hard line wrapping.
