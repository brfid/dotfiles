# yazi

Yazi uses a balanced three-pane layout, natural sorting, wrapped previews, `?`/`~` help bindings, and the gruvbox-dark flavor.

Flavor installed via `ya pkg add bennyyip/gruvbox-dark`.

Live paths:
- `yazi.toml` -> `~/.config/yazi/yazi.toml`
- `keymap.toml` -> `~/.config/yazi/keymap.toml`
- `theme.toml` -> `~/.config/yazi/theme.toml`
